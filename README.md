Koa application starter

## Setup

For install

```sh
bun install # or nci
```

For build

```sh
bun run build # or nr build
```

For lint

```sh
bun run lint # or nr lint
```

For test

```sh
bun test # or nr test
```

For all scripts

```sh
bun run all # or nr all
```
